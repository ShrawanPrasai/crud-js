# Single day to do list

A simple and efficient list application built with Vanilla JavaScript. This app allows users to manage their single day work with time scheduling features, ensuring an organized shopping experience.

## Features

- Add Items: Easily add work with a specific time.
- Edit Items: Update the name or time of existing items.
- Delete Items: Remove items from the list.
- Mark as Completed: Toggle items as completed/done.
- Auto-Sorting: Items are automatically sorted by time.
- Local Storage: Data is persisted in the browser's local storage, so you don't lose your list upon refresh.
- Responsive Design: Works well on different screen sizes.

## Technologies Used

- HTML5: For structure.
- CSS3: For styling (Modular CSS).
- JavaScript (ES6+): For application logic and DOM manipulation.
- FontAwesome: For icons.

## Project Structure

- `index.html`: Main entry point.
- `js/`: Contains JavaScript modules.
  - `app.js`: Main application logic and state management.
  - `form.js`: Handles form creation and submission.
  - `items.js`: Manages the list container.
  - `single-item.js`: Renders individual list items.
  - `data.js`: (Optional) Initial data structure reference.
- `css/`: Contains CSS files for different components.

## Usage

1.  Clone or download the repository.
2.  Open `index.html` in your web browser.
3.  Start adding items to your todo list!
